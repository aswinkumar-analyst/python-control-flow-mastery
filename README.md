# Python Control Flow Mastery

[![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)](https://www.python.org/)
[![Jupyter](https://img.shields.io/badge/Jupyter-Notebook-orange.svg)](https://jupyter.org/)

Interactive Jupyter Notebook demonstrating core Python control flow concepts: **for loops**, **while loops**, and **user-defined functions**.

## ✨ Features

- **For Loops**: Iterating over ranges and repeating tasks
- **While Loops**: Conditional repetition with input validation
- **User-defined Functions**: Modular code with loops and conditionals
- **Interactive Programs**: Real-time user input examples (guessing game, user verification)

## 📋 What You'll Learn

- Printing sequences with `for` loops
- Building interactive guessing games with `while` loops
- User input handling and validation
- Creating reusable functions
- Combining loops with conditional logic (`if-else`)

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Jupyter Notebook / VS Code with Jupyter extension

```bash
# Clone the repository
